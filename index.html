Pong Arcade Game on Microcontrollers

Grade: 100%

For a university project I developed a version of PONG in embedded C++ which ran on a pair of Arduino Nano microcontrollers. I designed and assembled a PCB to connect peripheral devices to the microcontrollers.
This project was significantly beyond the scope of the university assignment. I chose it specifically to challenge myself to learn outside the box.
Due to the video buffer taking up the majority of the 2KB available ram, the game was split between two microcontrollers. One of which handled video output while the other processed game logic and managed IO for the controllers, score counters, and audio.

Pixel Sorting Image Filter

Grade: N/A

As an exercise to learn C# multithreading I created an application which applies a visual effect to a bitmap.
The application first generates a contrast mask from the given image where all pixels exceeding a given brightness threshold are white and all others are black.
Contiguous lines of image pixels matching white pixels in the contrast mask are placed into groups and added to a queue to be sorted.
In a separate thread these groups are sorted by either hue, saturation, or luminance, then placed back into the image.

Maze Solving Algorithm and Animation

Grade: 100%

This project represents a maze as a graph where each node holds connections to up to two other nodes.
Maze solving is implemented using a recursive depth first search of the graph.
This algorithm is slowed for display as an animation which runs in a separate thread. Without delays it completes in approximately 1ms.
